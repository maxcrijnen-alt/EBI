# Source Ingestion Workspace

This folder is reserved for later import of course materials.

- `raw`: uploaded PDFs and DOCX files
- `processed`: extracted text and normalized chunks
- `questions`: reviewed question records ready for import
- `summaries`: topic summaries and study notes
- `formulas`: extracted formula records
- `rubrics`: grading rubrics for open-ended answers

Version 1 registers the known source files in the database but does not copy official questions into the generated practice bank.
